# 卡片內背景圖（直式卡片）

背景圖會顯示在**中央直式卡片裡面**（不是整個螢幕外框）。

請將照片放在此資料夾：

| 檔名 | 頁面 |
|------|------|
| `home.jpg` | 首頁 `/` |
| `intro.jpg` | 測驗說明 `/quiz` |
| `play.jpg` | 作答 `/quiz/play` |
| `result.jpg` | 結果頁 |

建議使用**直式**構圖（例如 9:16），與手機卡片比例較合拍。

程式會依序嘗試 `.jpg` → `.jpeg` → `.png` → `.webp`（`intro` / `result` 另有 `.svg` 後備）。
