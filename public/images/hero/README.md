# 主視覺圖（卡片內，非全螢幕背景）

| 檔名 | 使用位置 |
|------|----------|
| `hero.jpg` | 測驗說明頁 `/quiz` |
| `result-oily.jpg` | 油頭結果頁 |
| `result-sensitive.jpg` | 敏感結果頁 |
| `result-dry.jpg` | 乾性結果頁 |
| `result-neutral.jpg` | 中性結果頁 |
| `result-thin.jpg` | 髮量少結果頁 |
| `result-hairloss.jpg` | 掉髮結果頁 |

放入 JPG 後，在對應頁面或 `HeroImageSlot` 的 `src` 改為新路徑即可。
